### 🖋️ 자기소개

저는 고성능 애플리케이션과 확장 가능한 디지털 생태계 구축에 열정을 가진 **역동적이고 성장 지향적인 소프트웨어 엔지니어**입니다. 기술적 정확성과 창의적 사고를 결합하여 코드 한 줄 한 줄에 탁월함을 담아내고자 노력합니다.

- 🏗️ **핵심 역량:** **풀스택 개발**, **ERP 시스템 (Odoo)** 및 **엔터프라이즈 아키텍처** 전문가.
- 📱 **모바일 혁신:** **Flutter** 및 **React Native**를 활용한 원활한 크로스 플랫폼 경험 구축.
- ⚙️ **전통과 미래:** **EJB/Swing** 시스템을 현대적인 **클라우드 네이티브** 시대로 전환.
- ✨ **목표:** 뛰어난 엔지니어링을 통해 비전을 현실로 구현.
